## Sharing ami id to one region to another and one account to another account

#  1 step:- take the ami backup ( note:- make sure while taking the backup no reboot option shoud select)

         Note:- after completed the ami backup it should available then only we can share

# 2 step:- select ami id then click actions> copy ami > select the region > then save

# 3 step:- select ami id > then click edit permissn options> add account number > save 
                   note:- make sure below option must select
                   Add 'Create volume' permission to associated snapshots when creating account permissions.
